ช่วยเหลือ

หากคุณพบปัญหาในการใช้งานอื่นที่ไม่มีในคำถามด้านล่างนี้ หรือต้องการติชม, เสนอแนะ สามารถติดต่อเราได้โดยกรอกข้อมูลในแบบฟอร์มนี้

คำถามที่พบบ่อย
  • โหลดข้อมูลการจราจรไม่ได้
    หากคุณใช้งานตัวบล็อกโฆษณา ลองปิดใช้งานตัวบล็อกโฆษณา และโหลดหน้าเว็บใหม่อีกครั้ง หรือแจ้งเราผ่านฟอร์มติดต่อในลิงก์ด้านบน
  • เพิ่มเป็นแอปบนโทรศัพท์มือถือ
    สำหรับ Android เปิดเว็บนี้ผ่าน Chrome จากนั้นกดปุ่มเมนู และเลือก "เพิ่มไปยังหน้าจอหลัก"

    สำหรับ iOS เปิดเว็บนี้ผ่าน Safari จากนั้นกดปุ่มแชร์ และเลือก "เพิ่มไปยังหน้าจอโฮม"

    หากคุณเปิดดูข้อมูลใดอยู่และกดเพิ่มเป็นแอป เมื่อคุณเปิดแอปจะโหลดเนื้อหานั้นให้โดยอัตโนมัติ ดูเพิ่มเติมที่หัวข้อ "ลิงก์ไปยังเนื้อหา"
  • ใช้งานผ่าน HTTPS
    RainRadar บังคับการใช้งานผ่าน HTTPS ตั้งแต่เวอร์ชัน 7.0.0 (22 ก.พ. 2561)

    ในการใช้งานผ่าน HTTPS เมื่อคุณเรียกดูภาพเรดาร์ รูปภาพจะถูกดาวน์โหลดไว้บนเซิร์ฟเวอร์ก่อนจะส่งให้คุณ ดังนั้นรูปภาพอาจใช้เวลาโหลดนานกว่าในการใช้งานผ่าน HTTP

    ในการใช้งานผ่าน HTTPS ภาพเรดาร์แบบเคลื่อนไหวจะไม่สามารถใช้งานได้ เนื่องจากใช้แบนด์วิดทฺสูง โดยจะแสดงรายการลิงก์สำหรับเปิดเรดาร์ผ่าน HTTP แทน
  • โหลดภาพเรดาร์ไม่ได้, ภาพเรดาร์ไม่สมบูรณ์, ภาพเรดาร์ไม่ใช่ภาพล่าสุด
    เรดาร์หรือเซิร์ฟเวอร์ภาพเรดาร์อาจมีปัญหาชั่วคราว ลองเลือกดูเรดาร์อื่นแทน หรือแจ้งเราผ่านฟอร์มติดต่อในลิงก์ด้านบน
  • เวลาในภาพเรดาร์
    เวลาในภาพเรดาร์ของสำนักการระบายน้ำฯ เป็นเวลาท้องถิ่นในประเทศไทย

    เวลาในภาพเรดาร์ของกรมอุตุนิยมวิทยา และกรมฝนหลวงฯ เป็นเวลามาตรฐานสากล ซึ่งช้ากว่าประเทศไทย 7 ชั่วโมง ตัวอย่างเช่นเวลาในภาพเรดาร์คือ 15:30 น. จะเท่ากับ 22:30 น. ในประเทศไทย
  • รัศมีของเรดาร์
    • เรดาร์ของสำนักการระบายน้ำ กทม. เรดาร์หนองบอนมีรัศมี 50 กม. / เรดาร์หนองจอกมีรัศมี 90 กม. / เรดาร์หนองแขมมีรัศมี 150 กม.
    • เรดาร์ทั้งหมดของกรมอุตุนิยมวิทยามีรัศมี 240 กม. และมีเรดาร์บางส่วนที่มีรัศมี 120 กม. ร่วมด้วย
    • เรดาร์ทั้งหมดของกรมฝนหลวงฯ มีรัศมี 240 กม.
  • ลิงก์ไปยังเนื้อหา
    เมื่อคุณเลือกดูเรดาร์หรือข้อมูลใดๆ ก็ตาม URL ในเบราว์เซอร์ของคุณจะเปลี่ยนไป คุณสามารถใช้ URL ดังกล่าวเพื่อเปิดหน้าเว็บที่แสดงเนื้อหานั้นๆ ได้, คุณสามารถบุ๊กมาร์ก URL ไปยังเรดาร์หรือข้อมูลที่คุณดูบ่อย หรือแบ่งปันให้เพื่อนได้

    ตัวอย่าง: radar.pknme.com/#traffic จะแสดงข้อมูลการจราจรจากทวิตเตอร์
  • ข้อมูลการจราจรที่แสดงน้อยเกินไป ต้องการดูเพิ่มเติม
    กดปุ่ม "โหลดทวีตเพิ่มเติม" ด้านล่างสุดเพื่อโหลดข้อมูลเพิ่ม
  • มีอะไรใหม่?

    ด้านล่างนี้เป็นการเปลี่ยนแปลงที่น่าสนใจจากอัปเดตล่าสุด คลิกที่นี่เพื่อดูทั้งหมด

    RainRadar เวอร์ชัน 7.1.1 อัปเดตเมื่อ 22 มี.ค. 2561

    • ข้อมูลคุณภาพอากาศจาก aqicn.org ในหน้าแรก
    • บังคับการใช้งานผ่าน HTTPS (ดูรายละเอียดที่หัวข้อ "ใช้งานผ่าน HTTPS")
    • เพิ่มเรดาร์หนองบอน
    • เพิ่มประสิทธิภาพการโหลดข้อมูลต่างๆ
    • เปิดเนื้อหาจาก URL ได้ เช่น radar.pknme.com/#traffic จะแสดงข้อมูลการจราจรจากทวิตเตอร์ (ดูรายละเอียดที่หัวข้อ "ลิงก์ไปยังเนื้อหา")
